# 邏輯運算符 || 跟 ?? 的差別

## 結論

-   `??` 比 `||` 嚴謹
-   `??`是左邊只是`null`或是`undefined`，才會 return 右邊
    -   是只有`null`跟`undefined`，就算左邊是 `false`，也不會 return 右邊，挺有趣的，如下圖
    -   `null` 是 javascript 裡面，是`特殊的 object`，在某些狀況會被當成是 `truthy value`，如下圖
-   `||`左邊是`falsy value` 就會 return 右邊

-   -0
-   "" (空字串)
-   NAN
-   null
-   undefined
